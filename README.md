# SingTelThis is a repo for managing machine readable artifacts for the [SingTel APIs](http://six.sg/singtel-web-services). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [SingTel APIs](http://six.sg/singtel-web-services).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://six.sg/singtel-web-services)- [Developer](http://six.sg/singtel-web-services)- [Twitter](https://twitter.com/SingTel)- [GitHub](https://github.com/singtel)The [APIs.json](https://github.com/api-evangelist/singtel/blob/master/apis.json) index provides links to all the relevant links for the SingTel APIs, including the OpenAPI and Postman if they are present.
